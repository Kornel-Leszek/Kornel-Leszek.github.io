Zadanie 21 - ID i CSS

Pojęcie Znaczenie pojęcia
Co znaczy skrót ID? Identyfikator w HTML/CSS, który jest unikalny w danym dokumencie i służy do oznaczania pojedynczego elementu.
Jaki jest cel stosowania ID? Stosowanie ID pozwala na unikalne identyfikowanie elementów w dokumencie HTML, co umożliwia przypisanie do nich specjalnego stylu lub wykonanie akcji w JavaScript.
Jakie jest znaczenie znaku # Znak "#" przed nazwą identyfikatora oznacza, że styl lub akcja będą dotyczyć elementu o danym identyfikatorze.
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? Wywołanie ID następuje poprzez zastosowanie selektora w arkuszu stylów CSS, np. #nazwa_id, gdzie 'nazwa_id' to unikalna nazwa przypisana do elementu HTML.
Uwagi na temat stosowania ID (trzy konieczne wyliczenia) 1. ID musi być unikalne w całym dokumencie.
2. ID nie może być przypisane do wielu elementów.
3. Znak # służy do wskazania ID w CSS i JavaScript.
