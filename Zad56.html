1) Składnia teoretyczna definiowania klasy

selektor.nazwa_klasy { cecha: wartość; }

2) Opis dla: Klasa "cecha" oraz "wartość"

Klasa to wartość atrybutu class="...", nadana znacznikowi HTML.
Cecha to właściwość CSS (np. color, font-size).
Wartość to przypisanie cechy, np. red, 14pt.

3) Jaki znacznik może być selektorem? Trzy przykłady selektorów dla klas (ze znaczeniem)

  • p.klasa – akapit z przypisaną klasą
  • h1.tytul – nagłówek z przypisaną klasą
  • td.komorka – komórka tabeli z przypisaną klasą

4) Uwagi na temat nazwy klas

Nazwą może być dowolny pojedynczy wyraz, nie może on zawierać:
a) spacji
b) kropek
c) przecinków
h) znaku równości

Może zawierać:
a) litery (A-Z, a-z)
b) cyfry (0-9)
d) myślniki (-) i podkreślniki (_)

Nazwa nie może zaczynać się od cyfry ani myślnika.
Gdy chcemy użyć "zakazanych" znaków, należy poprzedzić je odwróconym ukośnikiem (\).

5) Jakie jest zastosowanie klas?

Klas używa się do nadawania stylów wybranym elementom HTML, szczególnie gdy te same style są wielokrotnie stosowane.

6) Kiedy ma zastosowanie lista klas?

Gdy chcemy przypisać kilka różnych stylów jednocześnie do jednego elementu.

7) Składnia teoretyczna listy klasy + uwagi

<selektor class="klasa1 klasa2 klasa3">...</selektor>

Uwagi:
– Wszystkie klasy muszą być zdefiniowane w arkuszu stylów.
– Kolejność nazw nie wpływa na wygląd.
– Jeśli kilka klas definiuje tę samą właściwość, decyduje zasada kaskadowości.

8) Czym są podzbiory klas + składnia

Podzbiory klas umożliwiają zastosowanie stylu tylko dla elementów mających jednocześnie kilka klas.

selektor.klasa1.klasa2 { cecha: wartość; }

9) Co jest sygnałem, że należy zastosować klasy?

Powtarzalne formatowanie na wielu stronach to sygnał, że należy użyć klas – dla ułatwienia modyfikacji i zachowania spójności.

10) Kiedy i z jakiego powodu powinny być planowane klasy dla wykonywanej strony?

Już na etapie projektu – dla oszczędności czasu, ułatwienia stylizacji i późniejszych zmian.

11) Jaki jest cel stosowania komentarzy podczas definiowania klas?

Komentarze (/* ... */) pomagają opisać przeznaczenie klas i zwiększają czytelność arkusza stylów.

12) Przykład definiowania klasy i jej użycia

CSS:

p.przyklad {
  color: red;
}

HTML:

<p class="przyklad">To jest akapit z klasą.</p>

13) Klasa odnosząca się tylko do jednego selektora + przykład

CSS:

p.moja {
  color: green;
  font-size: 14pt;
}

HTML:

<p class="moja">Treść akapitu</p>

14) Cel stosowania klasy uniwersalnej. Znaczenie „*”. Przykład

Cel: umożliwia użycie tej samej klasy w różnych znacznikach.
„*” – oznacza dowolny znacznik (uniwersalny selektor).

CSS:

*.moja {
  color: red;
}

Lub bez „*” (zalecane):

.moja {
  font-size: 12pt;
  color: red;
}

HTML:

<p class="moja">Akapit</p>
<h1 class="moja">Nagłówek</h1>
<ol class="moja"><li>Lista</li></ol>
