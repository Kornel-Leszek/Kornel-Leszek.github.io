Pseudoelementy w CSS

:first-line

Pseudoelement ::first-line pozwala na zastosowanie stylów tylko do pierwszej linii tekstu w elemencie blokowym.

Składnia teoretyczna:

selektor::first-line {
  cecha: wartość;
}

Przykład:

p::first-line {
  font-weight: bold;
  color: darkblue;
  font-size: 18pt;
}
Kolejnym pseudoelementem jest :first-line, który w przeciwieństwie do poprzednika obejmuje formatowaniem całą pierwszą linię, a nie tylko pierwszą literę.

:before i :after

Pseudoelementy ::before i ::after umożliwiają wstawienie dodatkowej treści przed lub po zawartości elementu.

Składnia teoretyczna:

selektor::before {
  content: "tekst lub znak";
  cecha: wartość;
}

selektor::after {
  content: "tekst lub znak";
  cecha: wartość;
}

Przykład:

p::before {
  content: "» ";
  color: grey;
}

p::after {
  content: " «";
  color: grey;
}
To jest przykład elementu z ::before oraz ::after.

:first-letter

Pseudoelement ::first-letter pozwala na stylizację pierwszej litery elementu blokowego (np. akapitu).

Składnia teoretyczna:

selektor::first-letter {
  cecha: wartość;
}

Przykład:

p::first-letter {
  font-size: 113pt;
  color: silver;
  font-weight: bold;
  float: left;
  margin-right: 8px;
}
Programowanie jest podstawą nowoczesnych technologii cyfrowych.
