1. W jaki sposób można określić właściwość danego obiektu?

Właściwości (pola) obiektu można określić na dwa sposoby:

  1. Bezpośrednio podczas tworzenia obiektu – przy użyciu zapisu w nawiasach klamrowych:
let samochod = {
    marka: "Toyota",
    kolor: "Czerwony"
};
  1. Dynamicznie po utworzeniu obiektu – dodając nowe właściwości w kodzie:
samochod.rok = 2020;
samochod['przebieg'] = 15000;

W obu przypadkach tworzymy lub modyfikujemy właściwości obiektu.

2. W jaki sposób można odwołać się do pól (właściwości) i metod?

Do pól (właściwości) i metod obiektu odwołujemy się za pomocą operatora kropki (.) lub nawiasów kwadratowych.

console.log(samochod.kolor);      // dostęp przez kropkę
console.log(samochod['marka']);   // dostęp przez nawiasy
samochod.start();                 // wywołanie metody

3. Cechy charakterystyczne konstruktora

  • Jest to specjalna funkcja używana do tworzenia nowych obiektów.
  • Jej nazwa zwykle zaczyna się wielką literą.
  • Używa się jej razem ze słowem kluczowym new.
  • Może inicjalizować pola (właściwości) obiektu.
  • Może zawierać metody przypisywane obiektowi.

4. Konstruowanie obiektu – przykład

let osoba = {
    imie: "Jan",
    nazwisko: "Kowalski",
    wiek: 25,
    przedstawSie: function() {
        console.log("Cześć! Nazywam się " + this.imie + " " + this.nazwisko);
    }
};

osoba.przedstawSie();

Obiekt osoba posiada trzy pola (imie, nazwisko, wiek) oraz jedną metodę przedstawSie().

5. Znaczenie słowa this. Jak oddzielamy pola oraz metody?

Słowo kluczowe this odnosi się do bieżącego obiektu, w którym się znajduje. Używa się go, aby uzyskać dostęp do jego pól i metod.

Pola oraz metody w obiekcie oddzielamy przecinkami.

let komputer = {
    marka: "Dell",
    model: "XPS",
    pokaz: function() {
        console.log(this.marka + " " + this.model);
    }
};

6. Konstruktor w JavaScript – przykład

function Samochod(marka, kolor) {
    this.marka = marka;
    this.kolor = kolor;
    this.info = function() {
        console.log("Samochód: " + this.marka + ", kolor: " + this.kolor);
    }
}

let auto1 = new Samochod("BMW", "Czarny");
auto1.info();

W powyższym przykładzie Samochod to konstruktor, który tworzy nowe obiekty z właściwościami marka i kolor.

7. Znaczenie słowa kluczowego new

Słowo kluczowe new tworzy nowy obiekt na podstawie funkcji konstruktora. Automatycznie ustawia wartość this na nowo utworzony obiekt. 

let auto2 = new Samochod("Audi", "Srebrny");

8. Znaczenie słowa kluczowego prototype

Każda funkcja w JavaScript ma właściwość prototype. Służy ona do dodawania metod i właściwości, które będą wspólne dla wszystkich obiektów utworzonych z danego konstruktora. 

Samochod.prototype.zmienKolor = function(nowyKolor) {
    this.kolor = nowyKolor;
    console.log("Nowy kolor: " + this.kolor);
}

auto1.zmienKolor("Czerwony");

Prototype pozwala na dodawanie funkcji do wszystkich obiektów danego typu bez potrzeby modyfikacji ich definicji.
