1. Co to są pseudoklasyk?

CSS przewiduje kilka ściśle określonych klas pozwalających na formatowanie niektórych elementów dokumentu – są to pseudoklasy.

2. Uwaga dotycząca dwukropka:

Szalenie istotnym elementem w tej konstrukcji jest dwukropek występujący pomiędzy selektorem a nazwą klasy.

3. Tabela opisująca pseudoklasy:

Pseudoklasa Opis
a:link Wygląd hiperłącza, jeżeli nie zostało jeszcze odwiedzone.
a:visited Formatowanie hiperłącza już odwiedzonego.
a:hover Zmiany wyglądu po najechaniu kursorem (np. na odsyłacz).
a:active Styl dla elementu w chwili kliknięcia (przed puszczeniem przycisku myszy).
a:focus Styl dla elementu, który uzyskał fokus (np. po naciśnięciu TAB).
:lang Formatowanie dokumentów wielojęzykowych.
:first-child Formatowanie pierwszego dziecka danego elementu.

4. Zastosowanie pseudoklasy :active

1) Może być używana jako uzupełnienie definicji odnośników.
2) Może poprawiać czytelność formularzy poprzez wizualne wskazanie aktywnego elementu.

5. Przykłady (trzy) kodów definiowania pseudoklas (wraz z tematem):

Temat: Arkusz wykorzystujący :link oraz :visited

a:link {
    color: red;
}
a:visited {
    color: green;
}

Temat: Formatowanie wyglądu wszystkich odnośników w HTML

a {
    font-size: 12pt;
    text-decoration: none;
}
a:link {
    color: navy;
}
a:visited {
    color: green;
}
a:hover {
    color: red;
    text-decoration: underline;
}

Temat: Zastosowanie :focus

a:focus {
    color: brown;
}
